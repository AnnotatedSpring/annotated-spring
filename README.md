# Annotated Spring

Web site for Spring Screencasts

## Tests

To run the test suite, run Maven with the `test` profile:

    $ mvn -P test

Or you can simply just run `mvn`.

## Development Server

To run a local development server, run Maven with the `development` profile.

    $ mvn -P development
